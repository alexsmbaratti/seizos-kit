# SeizosKit

SeizosKit is a Swift package that provides reusable Swift code for building robust and consistent apps across iOS, macOS, watchOS, and visionOS. 

## Platforms
* iOS 17+
* macOS 15+
* watchOS 11+
* visionOS 1+

## Installation
Add SeizosKit to your project using Swift Package Manager:

1. In Xcode, go to File → Add Packages…

1. Enter the repository URL:
`https://github.com/alexsmbaratti/seizos-kit`

1. Select the package products you want to include:
* SeizosUI (UI components)
* SeizosCore (logic/core types)
